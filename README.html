Requirements

  1. mongodb
  2. download and install the mongdb

  3. write a conf file, the content is below:

    #mongodb.conf: port=27017 logpath=/var/log/mongodb.log logappend=true dbpath=/var/data/mongodb fork=true

  4. start the mongo server

    $mongod -f mongodb.conf

  5. pip (pip is python package manager, you should install python 2.7 firstly)
  6. install pip through the command below

    curl -O https://bootstrap.pypa.io/get-pip.py python get-pip.py

  7. tornado, motor
  8. install the 3-party python requirements

    $pip install 'tornado==4.0.1' $pip install 'motor==0.3.2'

Hwweb(Homework web system) deployment

  1. import the data to mongodb

    $cd data_script $./dataImport

  2. modify the port running on the localhost(optional)
  3. the default port is 8888, you can change it in the index.py
  4. run the web system

    $ screen python index.py

  5. use ngrok to publish your app on the public network
  6. Download ngrok and install it.
  7. To use a solid domain, you need sign up the ngrok.com and get the right to bind your domain with your localhost port.
  8. You can register a free domain, such as tk domain.
  9. Assuming you have gotten a domain (eg: http://www.ucas-2014.tk) and got a token(eg:iuhnCIczc1NcieEXCs3Q) belonging to you when signing up on the https://ngrok.com . You run your app on localhost through 8888 port. You can bind your local app with the public domain through the commands below:

    $ngrok -authtoken iuhnCIczc1NcieEXCs3Q $screen ngrok -hostname ucas-2014.tk 8888
